# Steve Tietze — Location Timeline + Signature Work (with years)

## 1970-1980 | Munich, Germany -> (Move to US at ~10)

Lived: Munich, Germany (born 1970); moved to the United States around age 10.

Notable contributions / "origin story" projects: early programming + graphics experiments (Atari 400 -> VIC-20 -> C64), manually typing code from BYTE Magazine; sprite graphics and tinkering mindset established.

## 1980s-early 1990s (approx) | Kansas (Kansas State University)

Lived: Kansas (KSU).

Notable contributions: built strong Unix foundations using SPARC workstations running BSD Unix; early 3D/graphics experiments; active in Amiga BBS/newsgroups; created simple rotating 3D logos for the university.

## Early 1990s (approx) | Santa Cruz / Scotts Valley, California

Lived: Santa Cruz area (Scotts Valley scene).

Signature projects / "cool things":

Amiga + Video Toaster + LightWave 3D era: serious 3D modeling/animation growth.

Built a hardware/software hack: Amiga controlling a laserdisc player via RS232 to recreate scenes from Dragon's Lair (a personal R&D milestone blending physical/digital systems).

Social proximity to Silicon Valley pioneers; learned directly from Jay Miner (Amiga creator); crossed paths with Jobs/Woz/Bushnell in that community.
